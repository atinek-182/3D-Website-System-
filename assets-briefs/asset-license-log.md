# Asset License Log

Use this file to log details, licensing, and optimization choices for all external 3D models, textures, images, SVGs, and video assets used in the project.

---

## Logging Guidelines

- **Prohibited Assets**: Copyrighted trademarked brand/IP assets, weapons, adult material, gambling assets, or items with vague/unknown licenses.
- **Preferred Sources**: CC0/Free providers like Poly Haven, ambientCG, Quaternius, Kenney, or Khronos samples.
- **Log immediately**: Add a new log entry using the template below before integrating any downloaded asset into the code.

---

## Log Template

```markdown
## [Asset Name]

- **Source**: [e.g. Poly Haven, ambientCG, Custom, etc.]
- **URL**: [Direct link to asset download]
- **License**: [e.g. CC0, CC-BY 4.0 (attribution required), etc.]
- **Attribution Required**: [If yes, specify exact text/author link to display on page]
- **File Format**: [e.g. GLB, GLTF, PNG, WebP, WebM]
- **Polycount / Texture Size**: [e.g. 15,000 tris, 1K textures]
- **Optimization Done**: [e.g. DRACO compressed, downscaled textures from 4K to 1K, converted PNG to WebP]
- **Used In**: [e.g. components/scene/BackgroundMesh.tsx]
- **Keep/Remove Decision**: [e.g. Keep - approved for main release]
```
