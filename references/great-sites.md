# Great Website References

## Purpose

This file stores websites I like and converts them into reusable design principles for future 3D website projects.

## How To Use This File

* Add only real websites I personally like.
* Analyze the exact parts I like.
* Do not write vague lines like "make it premium".
* Convert each reference into clear rules.
* Future agents must read this file before creating visual direction.

## Global Reference Rules

1. References are for extracting principles, not copying designs.
2. Do not copy layouts directly.
3. Do not copy brand identity.
4. Do not copy text, assets, or visuals.
5. Extract:
   * layout principle
   * typography principle
   * motion principle
   * interaction principle
   * 3D/canvas principle
   * color/lighting principle
   * performance lesson
6. Every reference must include:
   * what I like
   * what I dislike
   * what to adapt
   * what to avoid
7. If a reference is not analyzed, do not use it as a design rule.
8. If a reference is old or changed, mark it for re-review.
