# SVG And Illustration References

## Purpose

This file stores SVG, icon, line-art, technical diagram, cursor graphic, grid overlay, and animated path references.

## Global Reference Rules

1. References are for extracting principles, not copying designs.
2. Do not copy layouts directly.
3. Do not copy brand identity.
4. Do not copy text, assets, or visuals.
5. Extract:
   * layout principle
   * typography principle
   * motion principle
   * interaction principle
   * 3D/canvas principle
   * color/lighting principle
   * performance lesson
6. Every reference must include:
   * what I like
   * what I dislike
   * what to adapt
   * what to avoid
7. If a reference is not analyzed, do not use it as a design rule.
8. If a reference is old or changed, mark it for re-review.

## Good SVG Rules

* clean paths
* scalable viewBox
* minimal metadata
* no random IDs
* accessible label if meaningful
* currentColor when useful
* optimized before use

## Animated SVG Rules

* GSAP for path drawing and timelines
* Motion for React state animation
* CSS for simple hover/fill changes
* reduced-motion fallback
* no heavy filters unless justified
