# Video And Motion Graphic References

## Purpose

This file stores video loop, hero background, loading screen, cinematic transition, product demo, and motion-graphic references.

## Global Reference Rules

1. References are for extracting principles, not copying designs.
2. Do not copy layouts directly.
3. Do not copy brand identity.
4. Do not copy text, assets, or visuals.
5. Extract:
   * layout principle
   * typography principle
   * motion principle
   * interaction principle
   * 3D/canvas principle
   * color/lighting principle
   * performance lesson
6. Every reference must include:
   * what I like
   * what I dislike
   * what to adapt
   * what to avoid
7. If a reference is not analyzed, do not use it as a design rule.
8. If a reference is old or changed, mark it for re-review.

## When To Use Video

* if it creates atmosphere better than static image
* if it explains product/motion
* if it supports story
* if file size can stay controlled

## When Not To Use Video

* decoration only
* weak mobile support
* huge file size
* distracts from typography
* fights with 3D
* stock-looking background

## Final Rule

Video must have a role, compression plan, poster, and fallback.
