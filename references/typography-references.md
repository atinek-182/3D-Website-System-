# Typography References

## Purpose

This file stores typography systems I like: font pairings, heading structures, editorial layouts, mono text usage, and responsive type behavior.

## Global Reference Rules

1. References are for extracting principles, not copying designs.
2. Do not copy layouts directly.
3. Do not copy brand identity.
4. Do not copy text, assets, or visuals.
5. Extract:
   * layout principle
   * typography principle
   * motion principle
   * interaction principle
   * 3D/canvas principle
   * color/lighting principle
   * performance lesson
6. Every reference must include:
   * what I like
   * what I dislike
   * what to adapt
   * what to avoid
7. If a reference is not analyzed, do not use it as a design rule.
8. If a reference is old or changed, mark it for re-review.

## Premium Typography Principles

* strong hierarchy
* controlled spacing
* readable body copy
* intentional font pairing
* restrained mono usage
* large type with breathing room
* contrast without chaos
* no default-looking system font stack unless intentional

## Bad Typography Patterns

* all text same weight
* oversized heading with weak layout
* random all-caps
* tiny gray paragraph text
* too many typefaces
* overused tech mono everywhere
* no responsive scaling
